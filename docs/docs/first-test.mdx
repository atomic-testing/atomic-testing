This walkthrough shows a minimal Atomic Test using React and the HTML component driver. The same structure works in Vue or Playwright as well.

## 1. Install packages

```bash
pnpm add @atomic-testing/react-18 @atomic-testing/component-driver-html
```

## 2. Create a small component

```tsx title="WelcomeButton.tsx"
import { useState } from 'react';

export function WelcomeButton({ name }: { name: string }) {
  const [clicked, setClicked] = useState(false);
  return (
    <div>
      <h1 data-testid='greeting'>Hello {name}!</h1>
      <button data-testid='welcome-btn' onClick={() => setClicked(true)}>
        {clicked ? 'Welcome!' : 'Click me'}
      </button>
    </div>
  );
}
```

## 3. Define the scene

```ts
import { HTMLComponentDriver } from '@atomic-testing/component-driver-html';
import { byDataTestId, ScenePart } from '@atomic-testing/core';

export const welcomeScene = {
  greeting: { locator: byDataTestId('greeting'), driver: HTMLComponentDriver },
  button: { locator: byDataTestId('welcome-btn'), driver: HTMLComponentDriver },
} satisfies ScenePart;
```

## 4. Write the test

```ts
import { createTestEngine } from '@atomic-testing/react-18';
import { WelcomeButton } from './WelcomeButton';

it('welcomes the user when clicked', async () => {
  const engine = createTestEngine(<WelcomeButton name="Alice" />, welcomeScene);
  expect(await engine.parts.greeting.getText()).toBe('Hello Alice!');
  await engine.parts.button.click();
  expect(await engine.parts.button.getText()).toBe('Welcome!');
  await engine.cleanUp();
});
```

That's it! You've created a fully functioning Atomic Test. Continue with the [Step-by-Step Tutorial](./tutorial) for a more complete example.
